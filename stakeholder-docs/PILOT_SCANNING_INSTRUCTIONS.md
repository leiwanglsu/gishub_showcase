# Pilot Scanning Instructions

## Purpose
This document provides stakeholder-facing scanning guidance for pilot operations.

## Phase Boundary
Current pilot scans support preliminary screening only. They are not certification-final evidence.

## Pre-Scan Checklist
- Confirm target site and scope for the pilot run.
- Confirm operator role and review role.
- Confirm capture context notes will be recorded.
- Confirm non-certification statement is included in all downstream communications.

## Capture Method Notes to Record
For each pilot scan, document:
- distance to target,
- whether a tripod was used,
- scan duration,
- scan angle,
- number of control references used.

Consistent capture-method logging improves comparability and interpretation quality.

## During Scan
- Move deliberately and avoid abrupt path changes.
- Minimize occlusions where practical.
- Avoid mixing very different capture styles within the same comparison set unless explicitly documented.

## Post-Scan Review
- Confirm that scan context notes are complete.
- Flag uncertain captures for follow-up rather than forcing high-confidence interpretation.
- Route outputs through the standard pilot review path before external sharing.

## Required External Message
All pilot outputs must carry this statement:

"Preliminary and non-certification. Final FEMA Elevation Certificate values require control/checkpoint validation and licensed professional review."
